  1. If already completed or pending, return (no undo, no double-click).

  6. On response: set `xpFloat = result.xpEarned` (XP float at 600ms per timing). Call `onComplete(result)`.
  7. Set `pending = false`.
  8. After 600ms: clear `showStamp` (keep it visible while XP floats).
- `handleIncrement()`:
  1. If completed or pending, return.
  2. Set `pending = true`.
  3. Call `incrementProgress(quest.id)` server action.
  4. On response: update local progress state. If `result.completed`, trigger stamp + XP float same as handleComplete. Call `onComplete(result)`.
  5. Set `pending = false`.
